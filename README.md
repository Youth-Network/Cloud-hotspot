# Cloud-hotspot
This is a Cloud WiFi Hotspot Software for Guest WiFi

Support  Mikrotik RouterOS - Hotspot & Ubiquiti Network - Guest Portal

Solutions:
Vouchers ,SMS(Cellphone Message),Member Account,Wechat via Wi-Fico
